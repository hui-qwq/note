# 🏃‍♂️ 一、跑步计划（核心减脂）

## 📌 每周结构（循环4周）

> 一周 5练2休（不需要天天跑）

### ✔ Day 1：轻松跑

- 30分钟慢跑
- 能说话的速度

### ✔ Day 2：间歇跑（燃脂加速）

- 跑2分钟 + 走1分钟 × 8–10组

### ✔ Day 3：休息 / 快走30分钟

### ✔ Day 4：慢跑

- 35分钟稳定跑

### ✔ Day 5：间歇跑

- 同Day2

### ✔ Day 6：慢跑 + 多走路

- 慢跑20分钟 + 步行20分钟

### ✔ Day 7：休息

------

## ⚠️ 关键点（很重要）

- 不追速度
- 不跑到吐
- 跑完不需要瘫
- “能坚持30天”比强度重要

------

# 🍱 二、饮食计划（决定你能不能变帅）

## 📌 核心原则（记住这3句）

> 少一点主食 + 足够蛋白 + 远离糖油

------

## ✔ 每天吃法模板

### 🥚 早餐（必须吃）

- 鸡蛋 2个
- 牛奶 / 无糖豆浆 1杯
- （可加：面包1片或小红薯）

👉 目的：防止暴食 + 提升代谢

------

### 🍚 午餐（正常吃，但控制）

- 米饭：**半碗～2/3碗**
- 肉：一掌心大小（鸡肉/牛肉/鱼）
- 蔬菜：随便吃

👉 规则：

> 饭可以吃，但不要“堆满碗”

------

### 🍗 晚餐（最关键）

- 蛋白质（鸡蛋/豆腐/鸡胸/鱼）
- 蔬菜
- 主食：**尽量少或不吃**

👉 重点：

> 晚上少碳水 = 肚子掉最快

------

### 🧃 加餐（可选）

- 饿：酸奶 / 水果
- 不饿：不吃

------

## ❌ 禁止项（30天尽量避免）

- 奶茶 / 含糖饮料
- 夜宵
- 油炸食品
- 零食随手吃

------

# 📉 三、30天变化节奏（现实预期）

## ✔ 第1周

- 身体开始“轻一点”
- 胃口变干净一点

## ✔ 第2周

- 坐下肚子稍微变软一点点
- 精神状态提升

## ✔ 第3周

- 腰围开始明显变化
- 脸变清爽（别人会开始感觉你“状态好”）

## ✔ 第4周

- 肚子明显收一点
- 下颌线开始出现
- 整体“看起来更利落”

------

# 💡 四、额外加成（强烈建议做）

这些会让效果直接+30%：

## ✔ 每天走路8000步以上

（很关键，比你想象更燃脂）

## ✔ 每天喝水 1.5–2L

（减少水肿，脸更干净）

## ✔ 睡眠 7小时+

（睡不好=肚子更难掉）
