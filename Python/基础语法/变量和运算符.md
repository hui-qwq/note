### 字符串

```python
name = '11'
name = "11"
name = """111
22
"""

print(name[0:2]) # 这种方式可以直接输出字串， 左闭右开
print(name[0:]) #默认到最后
print(name[:2]) #默认从开头
#如果用负数的话，就是从后往前数
#用正数就是从前往后数
print(name[:-1])
#其实还有第三个参数，步长
print(name[::2]) #步长不写默认1
print(name[::-1]) #写负数就会反着输出

print(name.find("sb",3, 5)) #在范围[3,5)查找"sb", 返回值找到的第一个"sb"的索引，没找到返回-1
print(name.index('b',3, 5)) #在范围[3,5)查找字符's'的索引，返回值是范围内找到的第一个位置，只能查字符，没找到报异常
name.count("sb", 3, 5) #范围查找字符数量 空间常量级， 复杂度O(n).要比KMP好
name.replace("sb","cb", 2) #把字符串内的"sb"换成"cb"
name.split(',', 1)，#以name里的','分割字符串，并指定只分割1次， 返回列表
```
