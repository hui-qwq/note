## 元组

是 Python 中一种有序、不可变的数据结构。

类似于列表（list），但是元组创建后不能修改其中的元素。

格式：

tuple_name = (元素1, 元素2, ...)

例如：

t = (1, 2, 3, 4)

print(t)

输出：

(1, 2, 3, 4)
2. 元组的特点
2.1 有序

元组中的元素有固定顺序，可以通过索引访问。
```py
t = ("apple", "banana", "orange")

print(t[0])
print(t[1])
```
输出：

apple
banana
2.2 不可变（重点）

元组创建后，不能修改元素。

例如：
```py

t = (1, 2, 3)
```
t[0] = 100

报错：

TypeError: 'tuple' object does not support item assignment

因为：

tuple ≠ list

列表：

```py
a = [1,2,3]
a[0] = 100
```

可以。

元组：
```py
b = (1,2,3)
b[0] = 100
```

不可以。

2.3 可以存储不同类型的数据
t = (1, "hello", 3.14, True)

print(t)

输出：

(1, 'hello', 3.14, True)
2.4 可以嵌套

元组里面可以放元组：

t = ((1,2), (3,4))

print(t[0])

输出：

(1,2)
3. 元组的创建
3.1 普通创建
t = (1,2,3)

print(type(t))

输出：

<class 'tuple'>
3.2 空元组
t = ()

print(t)

输出：

()
