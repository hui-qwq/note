# 列表

```python
li = [1, 3, 4, 'sb'] # 列表里可以装很多类型的元素
li.append(4) # 添加，往后加
li.extend('fort') # extend里必须是可迭代元素， 会将这个可迭代元素分开添加
li.insert(0, 'xb') #指定位置插入
1 in li #查询某个元素是否再列表里， 返回bool
del li[2] #删除指定元素
del li #删除列表
li.pop() #默认弹最后一个 
li.pop(1) #删除第1个元素
li.remove('sb') #删除列表中找到的第一个这个元素， 没找到会报错
li.sort() #排序， 元素类型不一样会报错 
```
还有很多用法和字符串一样的
